# Sistema para gerenciamento de tarefas

Gerado para controle em tempo real de tarefas da equipe.

## 📖 Descrição do projeto

Desenvolvido como requisito avaliativo da disciplina Software Engineering, do curso de Análise e Desenvolvimento de Sistemas, este projeto tem como objetivo colocar em prática conceitos fundamentais de engenharia de software.

A aplicação permite que pequenas equipes registrem, acompanhem e gerenciem tarefas em tempo real, facilitando o controle do Work in Progress (WIP) e apoiando a implementação de metodologias ágeis no dia a dia das organizações.

## 🏹 Funcionalidades

* Realizar cadastro de usários;
* Criar tarefas, descreve-las e categoriza-las por status;
* Realizar pesquisa e edições dos registros sobre os usuários e das tarefas.

### 📋 Pré-requisitos

Para visualizar esse sistema é necessário ter:

Prompt de comando (terminal/cmd): normalmente presentes nos sistemas operacionais, porém também pode ser utilizado nos terminais de IDE's ou editores de texto.

Python: Linguagem de programação utilizada para desenvolver o projeto.

## ⚙️ Testes

* [Github Actions] - Executa os testes automaticamente para cada pull request.
